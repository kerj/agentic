## MUI Pattern Recommendations
- For forms: Use Stack with spacing instead of manual margin
- For layouts: Prefer Grid over manual flexbox
- For cards: Always use Card > CardContent > CardActions hierarchy
- For spacing: Use theme spacing units (1-10), not pixel values
- For colors: Always reference theme.palette, never hardcode

Remember: The goal is high-quality, maintainable MUI code with no unnecessary dependencies. Be constructive but firm about MUI requirements.
